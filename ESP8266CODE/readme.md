# ESP8266 TCP客户端

ESP8266 NodeMCU固件，用于康复医疗仪表盘系统与上位机通信。

## 概述

本固件运行在ESP8266 NodeMCU上，作为TCP客户端连接上位机，接收上位机的控制命令。

## 功能

- WiFi连接（连接2.4G热点）
- TCP客户端连接上位机服务器
- 接收上位机命令（协议解析）
- LED状态指示
- 串口调试输出

## 硬件

- ESP8266 NodeMCU
- 工作电压：3.3V
- 连接2.4G WiFi热点

## 快速开始

### 1. 修改配置

打开 `mian.cpp`，修改以下配置：

```cpp
// WiFi配置
const char* WIFI_SSID = "你的WiFi名称";
const char* WIFI_PASSWORD = "你的WiFi密码";

// 上位机TCP服务器配置
const char* SERVER_HOST = "192.168.x.x";  // 上位机IP地址
const int SERVER_PORT = 8080;               // 端口
```

### 2. 烧录

1. 打开Arduino IDE
2. 安装ESP8266开发板支持：
   - 文件 → 首选项 → 附加开发板管理器网址：
     ```
     http://arduino.esp8266.com/stable/package_esp8266com_index.json
     ```
   - 工具 → 开发板 → 开发板管理器 → 搜索ESP8266 → 安装
3. 选择开发板：**NodeMCU 1.0**
4. 选择端口
5. 上传烧录

## 通信协议

### 数据包格式

```
[起始符][长度][类型][数据][校验和][结束符]
  0xAA    1字节  1字节  N字节   1字节   0x55
```

### 命令类型 (上位机 → ESP8266)

| 命令码 | 功能 |
|--------|------|
| 0x01 | 设置力控参数 |
| 0x02 | 获取状态 |
| 0x03 | 系统复位 |
| 0x04 | 参数自动辨识 |

### 数据类型 (ESP8266 → 上位机)

| 数据码 | 功能 |
|--------|------|
| 0x10 | 传感器/电机数据 |
| 0x11 | 状态数据 |
| 0x12 | 响应数据 |

### 设置力控参数命令格式

```
[0xAA][0x03][0x01][通道][值][校验和][0x55]

示例：设置通道L值为50
0xAA 0x03 0x01 0x4C 0x32 0xXX 0x55
     |  |  |   |    |
     |  |  |   |    +-- 值 (50)
     |  |  |   +------- 通道 ('L'=0x4C)
     |  |  +----------- 命令 (0x01)
     |  +-------------- 长度 (3)
     +----------------- 起始符 (0xAA)
```

## LED状态指示

| 状态 | LED行为 |
|------|---------|
| 未连接WiFi | 快闪 |
| WiFi已连，未连服务器 | 慢闪 |
| 已连接服务器 | 常亮 |

## 串口命令

通过串口监视器发送命令（波特率115200）：

| 命令 | 功能 |
|------|------|
| STATUS | 查看系统状态 |
| RESET | 重启设备 |
| HELP | 显示帮助 |
